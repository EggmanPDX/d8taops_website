Section 01

Colors

The palette is locked. Three core brand colors + deep navy + white. Functional UI colors are for state and data only — never decoration. The tinted surface (#f0f4f8) is used for alternating light sections.

Core Brand Colors

Brand Navy #081F5C --color-navy H1, H2, section headings, proof block bg
Deep Navy #040e2e --color-deep-navy Footer bg, darkest hero gradient stop
Brand Blue #0477BF --color-blue H3, eyebrow labels, nav links, stat numbers
Brand Cyan #048ABF --color-cyan Gradient endpoint, CTA gradient
Green Accent #3ecf8e --color-green-accent Proof block accents, button glow, agent active states
Body Text #333333 --color-body All body copy on light sections
White #FFFFFF --color-white Light section backgrounds, text on dark sections

Surface System (5 levels)

Page BG#FAFBFF--color-bgBase page background — never pure white
Surface Container#F5F7FC--color-surface-containerCard backgrounds, agent icon bg
Tinted Surface#f0f4f8--color-tinted-surfaceAlternating section bg (ticker, personas, D8:VIEW)
Surface High#EEF2F9--color-surface-container-highElevated surfaces, hovered states

Functional UI Colors (state & data only)

Success Green#22C55E--color-successPositive metrics, "After" states
Success Bright#26D07C--color-success-brightProgress bars, PASS status badges
Alert Red#FF5252--color-errorViolations, failures
Signal Yellow#FFD500--color-warningStale data, REVIEW status badges
Grey Dark#515151--color-grey-darkUI sub-headers, secondary text
Grey Mid#6B7280--color-grey-midMuted body text
Light Grey#F1F5F9--color-grey-lightTable headers, empty chart tracks

D8 Gradient (logo & select brand moments only)

Top: Brand gradient (#081F5C → #0577BE → #048ABF → #FFFFFF). Logo and select brand moments only.

Bottom: Hero gradient (linear-gradient(160deg, #0c1428 0%, #0f2560 45%, #081F5C 100%)). Hero section and proof block only.

Only animate transform and opacity. Never width, height, top, left, margin, or padding. Never transition-all. Always specify the property. The site uses --ease-reveal: cubic-bezier(0.22, 1, 0.36, 1) for all scroll-triggered reveals.

Section 12

Do's & Don'ts

✓ Do

  • Left-justify all text — except the closing CTA section (center-aligned, intentional)
  • Alternate White (#FFFFFF) and Tinted (#f0f4f8) sections — never stack the same type twice
  • Use at least --space-24 (96px) between major section breaks
  • Use --text-4xl in #0477BF for key stat metrics
  • Include all three interactive states: hover + focus-visible + active
  • Use semantic HTML: nav, main, section, article, aside, footer
  • Set font-display: swap on all custom fonts
  • Pair Navy-tinted shadows: rgba(8, 31, 92, ...)
  • Use specific numbers: 97% faster, $1.2M+, 100% coverage
  • Respect prefers-reduced-motion on all animations
  • Use --ease-reveal (0.22, 1, 0.36, 1) for scroll-triggered reveals
  • Use 14px radius (not full pill) for the nav pill

✗ Don't

  • Don't use hr or 1px borders to separate sections — use tonal shifts
  • Don't center-align text (exception: closing CTA section only)
  • Don't use pure grey shadows — always Navy-tinted
  • Don't use transition-all — specify the property
  • Don't animate layout properties: width, height, top, left, margin, padding
  • Don't use equal-column grids (4/4/4 or 3/3/3/3)
  • Don't over-animate — 2–3 orchestrated moments per page, max
  • Don't use pure #FFFFFF as a page background — use #FAFBFF
  • Don't use default Tailwind colors (indigo-500, blue-600, etc.)
  • Don't use 3D robots, glowing brains, Matrix code, or stock handshakes
  • Don't use: transform, leverage, cutting-edge, seamless, robust, powerful
  • Don't use placeholder text as a label substitute
  • Don't use a blue button background — the CTA is navy (#081F5C) with a glow
  • Don't ship without verifying WCAG AA (4.5:1 normal text, 3:1 large/UI)

Section 14

Copy Rules

Design and words are the same discipline. These rules govern all copy on every D8TAOPS page.

Write this way

  • Lead with outcome, not feature: "97% faster" beats "AI-powered audit agent"
  • One idea per sentence. Short sentences.
  • Use specific numbers: 97% faster, $1.2M+, 8 agents, 90 days
  • Active voice only
  • Write like explaining it at a coffee meeting

Never use these words

  • Transform
  • Leverage
  • Cutting-edge
  • Seamless
  • Robust
  • Powerful
